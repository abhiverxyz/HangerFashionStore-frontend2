# Frontend 2

Greenfield frontend for Hanger Fashion Store. Built in sync with Backend2. User types: Admin, Users, Brands.

**Status:** Done for now. Auth, app shell, concierge (intent, avatars, conversation title, message blocks), shared product UX, and design foundation are in place. Further phases (Find, Looks, Closet, Store, Brand, Edit, Profile, Landing, Brand admin) can be picked up later.
